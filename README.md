## How to install?
<ol>
    <li>
        <code>git clone https://github.com/Kimheang-Developer/github-user-app.git</code>
    </li>
    <li>
        <code>cd github-user-app</code>
    </li>
    <li>
        <code>npm install</code>
    </li>
    <li>
        <code>npm start</code>
    </li>
</ol>
